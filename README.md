# 📝 Python To-Do List

A simple command-line To-Do List application built using Python.

🚀 Features

- Add new tasks
- View all tasks
- Delete tasks
- Menu-driven interface
- User-friendly console application

🛠️ Technologies Used

- Python 3

 ▶️ How to Run

```bash
python todo.py
```

 📋 Sample Menu

```
========== TO-DO LIST ==========
1. Add Task
2. View Tasks
3. Delete Task
4. Exit
```

 📚 Concepts Used

- Python Lists
- Loops
- Conditional Statements
- User Input/Output
- List Methods (`append()`, `pop()`)
